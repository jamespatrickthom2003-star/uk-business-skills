## How to Use

1. Download the `.md` file for the skill you need
2. Place it in your Claude Code project directory (or `.claude/commands/`)
3. Invoke it by name — Claude Code reads the skill and follows its instructions

These are Claude Code custom slash commands. They work with any Claude Code setup.
